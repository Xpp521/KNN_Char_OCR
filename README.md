# 基于KNN算法的字符识别系统
### 1、简介
毕设程序。可针对手写字符进行识别。KNN算法借助OpenCV库，采用蛮力方式实现。

### 2、第三方库
- OpenCV
- numpy

### 3、系统特点
- 可连接摄像头，动态识别画面中的字符；
- 可将样本数据保存到本地，以便下次使用；
- 支持在运行过程中手动添加、标注样本数据。
